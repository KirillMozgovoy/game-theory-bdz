## Краткое описание задачи
Оптимизация распределения вычислительных ресурсов облачного провайдера в условиях неопределенности нагрузки.
*   **Игрок A (Провайдер):** Пакетный (Batch) / Потоковый (Streaming) режимы.
*   **Игрок B (Среда):** Равномерная / Пиковая / Ночная нагрузка.

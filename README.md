## Описание задачи
Рассматривается задача оптимизации распределения вычислительных ресурсов облачного провайдера.
*   **Игрок A (Провайдер):** стратегии — пакетный режим (Batch) или потоковый режим (Streaming)[cite: 6].
*   **Игрок B (Среда):** стратегии — различные профили клиентской нагрузки (Равномерная, Пиковая, Ночная)[cite: 6].

## Итог решения
*   **Оптимальная стратегия Провайдера (A):** $x^* = (3/8, 5/8)$[cite: 6]
*   **Оптимальная стратегия Среды (B):** $y^* = (0, 3/8, 5/8)$[cite: 6]
*   **Цена игры:** $v = 9.75$[cite: 6]

## Практический смысл
Для обеспечения гарантированной маржинальности на уровне **9.75 у.е.** (независимо от действий клиентов), провайдеру следует распределять мощности следующим образом:
*   **37.5%** — на поддержку пакетного режима обработки[cite: 6].
*   **62.5%** — на поддержку потокового режима[cite: 6].
